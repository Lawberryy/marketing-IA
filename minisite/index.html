Prérequis
1/ Avoir les commandes git fonctionnelle

Vous devrez vérifier que les commandes git sont installé dans votre terminal.
Pour vérifier il vous suffit d'ouvrir votre terminal ou votre invite de commande et de taper git --version

Si la version apparait tout est bon. Si la commande n'est pas reconnue il faudra l'installer.

Installation sur mac :

Faire brew install git

Installation sur Windows :

Faire choco install git


2/ Avoir un compte github

Du côté de github, si vous n'avez pas encore créé votre profil sur le site, je vous laisse consulter le site éducation pour vous inscrire : https://education.github.com/pack


3/ Générer et lier github à une clé ssh

(Les étudiants qui ont fait le setup windows à la rentrée ont déjà réalisé cette étape je crois).

Que ce soit sur windows ou sur mac, il faudra entrer dans un terminal la commande ssh-keygen. Au niveau des questions, vous pouvez laisser les réponses vide en appuyant sur entrer. Cette génération de clé est à faire avec l'ordi avec lequel vous allez travailler le cours de git.

Il faudra récupérer la clé générée en ouvrant le fichier id_rsa.pub dans un éditeur de code ou bloc-notes.
Sur mac le chemin de la clé est : /Users/votre_nom_d_utilisateur/.ssh/id_rsa.pub
Sur windows : C:Usersvotre_nom_d_utilisateur/.ssh/id_rsa.pub

Si vous ne trouvez pas le dossier .ssh sur mac, faire les commandes suivantes :
defaults write com.apple.finder AppleShowAllFiles -bool true
killall Finder

Si vous ne trouvez pas le dossier .ssh sur wondows : https://support.microsoft.com/fr-fr/help/4028316/windows-view-hidden-files-and-folders-in-windows-10


Ouvrez le fichier et copier la clé.
Sur github dans les settings ssh (https://github.com/settings/keys), faire new ssh key, coller la clé et donner lui un titre.

Objectifs
L'objectif est de comprendre et maitriser git, outil qui permet de versionner et faire des snapshots de ses projets tech. On le mettra en synergie avec github, plateforme web qui permet de stocker ces différentes versions dans le cloud.
